What makes this agentic

Bede's opener fired the moment Emma's session loaded — she never tapped "Start Lesson." The session is already in motion when she arrives.

No content library
There's no menu to browse, no video to pick, no exercise to select. Bede chose the approach (close-your-eyes visualization) and the Socratic question sequence entirely.
Tool cards are invisible actions
The amber hint card wasn't a UI button Emma tapped — Bede's AI decided to call offer_socratic_hint and the card appeared automatically. Emma just sees a nudge; she doesn't know it's a tool.
Grade 4 (3-5 stage)
  • text-base (16px) — readable but not K-2 oversized
  • Standard Socratic pacing
  • Mic + TTS available but not forced
  • Timer hidden until final 5 minutes

Their land — and their power to make the rules on it.

Section 6

Child Safety Architecture — Why Only Emma Can Be Emma

LAN Boundary No internet access required. Caddy TLS terminates on your home network only. No external DNS lookup, no cloud account, no app store distribution. The server never calls home.
Device Identity JWT tokens are fingerprinted to the issuing IP address and browser User-Agent at login. A token from Emma's iPad silently returns 401 on any other device — no error message, no hint.
PIN Authentication 4-digit numeric PIN, verified server-side with hmac.compare_digest (constant-time, no timing oracle). Brute-force locked after 5 failed attempts. PIN never stored in plaintext.
An unauthorized adult cannot authenticate as a child without all of the following:

Knowing the child's PIN  ·  Producing a voice match ≥ 0.72 confidence against the enrolled voiceprint  ·  Being on the LAN  ·  Sending requests from the registered device  ·  All simultaneously. Failing any layer returns a generic 401 with no indication of which check failed.

Voice Biometric Resemblyzer speaker embedding compared against enrolled voiceprint. Confidence threshold 0.72 (high). 3 attempts before parent-password override is required. Audio blob never leaves the LAN server — Resemblyzer runs on-device CPU.
Content Constraint Bede's system prompt is strictly Charlotte Mason / Catholic homeschool. ExfiltrationGuard middleware strips SSE chunks containing prompt-injection markers before they reach the client. No external tool calls, no web browsing, no file system access from the AI.
Parent Oversight & Session Limits Full conversation transcript visible to parent in real time. Encrypted audit log of every auth event and tool call. 2-hour hard session limit enforced server-side — no client-side bypass possible. Parent can end session remotely; child cannot extend past the limit.
All student data — voice profiles, session configs, narration scores, conversation logs — is stored as AES-256-GCM encrypted BYTEA.
The database provider never sees plaintext. MASTER_SECRET → KEK → DATA_KEY hierarchy; key material is never written to disk in cleartext.
